VBStats – Final Year Project

VBStats is my final year project, focused on building a web-based platform for collecting, storing, and analysing volleyball statistics at both team and player level.

The system is designed to support match tracking, performance analysis, and future AI-driven insights. The application follows a clean client-server architecture with a strong emphasis on scalability and usability.

Key Features
  • Match and player statistics tracking
  • RESTful API for data access
  • Modern, responsive frontend
  • Structured backend with persistent storage
Technologies Used
  • React.js (Frontend)
  • Golang (Backend)
  • PostgreSQL
  • Docker (Development & deployment)

Further enhancements planned include advanced analytics, visual dashboards, and AI-assisted insights.
