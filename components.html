Bootstrap Components & Dynamic Features

This page outlines the Bootstrap components and dynamic features of the website

Bootstrap Components

  • Navbar – Responsive navigation bar with collapsible mobile menu and anchor-based section navigation.
  • Cards – Used to present projects in a structured, visually consistent format.
  • Modals – Display detailed project information and confirmation messages without page reloads.
  • Carousel – Image carousel used in the hobbies section with overlay content and custom image handling.
  • Accordion – Organises education details by academic year in a compact, expandable format.
  • Badges – Highlight technologies and tools used in projects.
  • Buttons – Used consistently across navigation, actions, and calls to action.
  • Forms – Contact form with success modal.
  • Grid System – Used throughout the site for responsive layouts across different screen sizes.

Dynamic & Interactive Features

  • Single-Page Navigation – Navbar links scroll smoothly to sections using anchor-based navigation.
  • Modal Triggers – Project cards and contact form trigger modals using Bootstrap’s JavaScript API, and JavaScript used for succes modal for contact form.
  • Responsive Image Handling – Carousel images use object-fit and object-position to maintain consistent sizing.
